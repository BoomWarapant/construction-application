ข้อจำกัดความรับผิดชอบ: ผลลัพธ์ทั้งหมดเป็นเพียงการวิเคราะห์เบื้องต้น (Preliminary Analysis / Conceptual) เพื่อช่วยจัดทำ Checklist และตรวจประเด็นสำคัญเท่านั้น มิใช่รายการคำนวณโครงสร้างขั้นสุดท้าย การออกแบบจริงต้องเป็นไปตาม มยผ.1301/1302-61, มาตรฐานคอนกรีตเสริมเหล็ก, กฎกระทรวงควบคุมอาคารและกฎหมายแผ่นดินไหว พ.ศ.2564 และต้องผ่านการตรวจสอบลงนามโดยวิศวกรโยธาผู้ได้รับใบอนุญาต (สามัญ/วุฒิวิศวกร) ก่อนนำไปก่อสร้างจริง

3. ข้อมูลน้ำหนักบรรทุก

⚖️ น้ำหนักบรรทุก (kg/m²)

🧮 Load Combination

ℹ️
Mass Source สำหรับแผ่นดินไหวโดยทั่วไปใช้ Dead Load + Superimposed DL + สัดส่วน Live Load (มยผ. ใช้ 100%DL ; LL บางส่วนตามการใช้งาน)

4. ข้อมูลแรงแผ่นดินไหว

วิธีออกแบบสเปกตรัมการตอบสนอง (Response Spectrum) ตาม มยผ.1302-61 / ASCE 7-10 — เหมือนรายการคำนวณ Star 8

🌐 ความเร่งตอบสนองเชิงสเปกตรัมตามที่ตั้ง (Mapped) มยผ.1302-61

ℹ️
เลือก จังหวัด/อำเภอ แล้วโปรแกรมจะดึงค่า Ss, S1 ตาม มยผ.1302-61 ตารางที่ 1 ให้อัตโนมัติ (พิมพ์ทับแก้เองได้) — ค่าตั้งต้น อ.สารภี จ.เชียงใหม่ = 0.927 / 0.236

📐 ค่าความเร่งตอบสนองเชิงสเปกตรัมสำหรับออกแบบ (คำนวณอัตโนมัติ)

🏗️ ตัวประกอบโครงสร้าง & คาบการสั่น default คำนวณอัตโนมัติ · แก้ไขได้

ℹ️
ค่า Ie, R, Ω₀, Cd, Ct, x โปรแกรมตั้งให้อัตโนมัติตามประเภทการใช้งานอาคาร / ระบบรับแรง / ระดับความเหนียว (มยผ. ตารางที่ 6 และ 9) — พิมพ์ทับเพื่อแก้เองได้  
RC IMRF ≈ 5
T = Ct·H^x
มยผ. = 0.02·hsx

🔻 ความไม่สม่ำเสมอของอาคาร (Irregularity)
