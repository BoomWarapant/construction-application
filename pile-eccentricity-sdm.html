1. ข้อมูลโครงการ Project Information

ข้อมูลทั่วไปสำหรับหัวรายการคำนวณและเอกสารขออนุมัติ

2. วัสดุและพารามิเตอร์ออกแบบ (SDM) Materials & Design Parameters

ตัวคูณลดกำลัง φ ตาม วสท. 011008-21 / ACI 318-19 Table 21.2.1 — ดัด 0.90, เฉือน 0.85 (วสท. หน่วย ksc)
อ้างอิง: น้ำหนักบรรทุกเพิ่มค่า U = 1.4DL + 1.7LL (วสท. 011008-21) — ใช้ตัวคูณเฉลี่ย Pu/P ≈ 1.40–1.50 เมื่อไม่แยก DL/LL · φ ดัด = 0.90, φ เฉือน = 0.85

3. ข้อมูลฐานราก ตอม่อ และเสาเข็ม Footing, Pier & Pile Data

มิติตามแบบก่อสร้างเดิม — แกน X = ด้านยาว L, แกน Y = ด้านกว้าง B จุดกำเนิด (0,0) ที่ศูนย์กลางตอม่อ
กฎกระทรวงฐานรากฯ พ.ศ. 2566 ข้อ 22: ความหนาฐานราก ≥ 35 ซม. (เข็มแบกทาน) / ≥ 20 ซม. (เข็มเสียดทาน) · อมหัวเข็ม ≥ 15 ซม. · ระยะขอบเข็มถึงขอบฐาน ≥ Dp/2 · ระยะห่างเข็ม ≥ 3Dp (แนะนำ วสท.)

4. น้ำหนักบรรทุกที่ตอม่อ Loading (Service Loads)

น้ำหนักใช้งาน (Working Load) ที่ถ่ายลงฐานราก — โมเมนต์ภายนอกจากโครงสร้างส่วนบน (ถ้ามี)

5. พิกัดเสาเข็ม — ตามแบบ และผลสำรวจหน้างาน Pile Coordinates: Design vs As-Built Survey

พิกัด (x, y) เทียบศูนย์กลางตอม่อ (0,0) หน่วยเมตร · Δx, Δy = ระยะหนีศูนย์จากการสำรวจ (เครื่องหมาย +/− ตามแกน) · พิกัดจริง = ตามแบบ + Δ
ช่องสีเน้น (ส้มดินเผา) = ผู้ใช้ป้อนข้อมูล · พิกัดจริง X′,Y′ คำนวณอัตโนมัติ · เครื่องหมายตามแกน: ขวา/บน = บวก, ซ้าย/ล่าง = ลบ
วิธีสำรวจ: ทาบ template หาศูนย์เข็มจริง → ขึงเอ็นตามกริดไลน์ทิ้งดิ่งหาศูนย์ตอม่อ → วัด offset แยกเป็นเวกเตอร์ Δx, Δy พร้อมเครื่องหมาย

6. วิเคราะห์กลุ่มเสาเข็มเยื้องศูนย์ (หน้าตัดอสมมาตร) Asymmetric Pile Group Analysis

ทฤษฎีกลุ่มเสาเข็มอสมมาตร รวมผลคูณโมเมนต์ความเฉื่อย Ixy — Ri = P/N + m·x′i + n·y′i (สมมติฐานฐานรากแข็งเกร็ง)

7. แนวทางแก้ไขฐานราก Remedial Options

ระบบประเมินเสนอแนวทางตามผลวิเคราะห์ — ผู้ใช้เลือกวิธีที่จะใช้ออกแบบจริงได้
B — ขยายมิติฐานราก (Enlarged Footing)
C — คานยึดรั้ง / คานถ่ายน้ำหนัก (Strap Beam)
D — เสาเข็มแซม (Compensation Piles)
ป้อนพิกัดเข็มแซม (เทียบศูนย์ตอม่อ) — ระบบจะรวมเข็มแซมแล้ววิเคราะห์กลุ่มอสมมาตรใหม่ทั้งฐาน (สมมติเข็มทุกต้นมีสติฟเนสแนวแกนเท่ากัน)

8. ตรวจสอบกำลังฐานรากด้วยวิธีกำลัง (SDM) Footing Strength Check — SDM

ใช้มิติฐานรากและแรงปฏิกิริยาเข็ม "หลังการแก้ไขตามวิธีที่เลือก" — โมเมนต์ดัดที่ขอบตอม่อ, แรงเฉือนคาน, แรงเฉือนทะลุ ตาม วสท. 011008-21 / ACI 318-19
